    0 AND 0 = 0
    1 AND 1 = 1

2)  A XOR B ::
    0 XOR 1 = 1
    0 XOR 0 = 0
    1 XOR 1 = 1

    A XOR B : A B为不同逻辑值时，逻辑表达式值为真，否则为假
